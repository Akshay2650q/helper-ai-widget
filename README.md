# Helper AI Starter

Simple AI helper app:

- **Backend**: Node.js + Express (calls OpenAI API)
- **Frontend**: React + Vite chat widget
- **Loader**: Embeddable widget script

## Setup

1. **Backend**:
   ```bash
   cd backend
   cp .env.example .env
   # Add your OpenAI API key to .env
   npm install
   node index.js
   ```

2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. Open frontend at `http://localhost:5173`
